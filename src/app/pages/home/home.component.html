¡Bienvenido!

La siguiente página web es una herramienta que busca apoyar al proceso de evalación de usabilidad web, la cual por medio de evalaciones de forma manual por parte de expertos. La aplicación web está enfocada en la evaluación de la usabilidad de sitios web, basándose en principios de usabilidad que serán seleccionados como parte del desarrollo del presente proyecto. Para el desarrollo de la aplicación web se tomaron en cuenta factores como:

  • - Investigación de los principios de usabilidad que existen para tener un amplio conocimiento de ellos y saber cuáles se van a considerar en para la aplicación web.
  • - Creación de un formulario con base en los principios de usabilidad seleccionados.
  • - Que en la evaluación se considere como mínimo la URL del sitio web a evaluar, el nombre del sitio web, información relacionada con el experto que evalúa, y preguntas relacionadas con la usabilidad web basadas en los principios que se seleccionen.
  • - Que haya una opción para gestionar la información de los principios de usabilidad.
  • - Una opción que permita generar un reporte tipo resumen con los resultados de la evaluación de usabilidad.
  • - Pruebas funcionales con las cuales el software será puesto a prueba para determinar si cumple con el alcance de los requerimientos establecidos.
  • - Dado el tiempo que se tiene para el desarrollo del presente trabajo de grado, no se contemplan pruebas con usuarios finales.
